# Harmony Music - 需求拆解文档

## 产品概述

- **产品类型**: PC端音乐播放器（Web原型）
- **场景类型**: <scene_type>prototype-app</scene_type>
- **目标用户**: 追求沉浸式听觉体验、偏好深色主题与现代美学设计的音乐爱好者
- **核心价值**: 打造集美观、智能、高效于一体的PC端音乐播放器，提供沉浸式音乐体验
- **界面语言**: 中文
- **主题偏好**: user_specified（深色主题 + 深紫到青色渐变）
- **导航模式**: 路径导航
- **导航布局**: Sidebar（左侧侧边栏导航 + 底部播放栏）

---

## 设计规范（来自设计文档）

### 色彩规范
- **主色调**: `#6366F1`（深紫色）→ `#06B6D4`（青色）渐变
- **背景色**: `#0F172A`（深色背景）、`#1E293B`（卡片背景）
- **文字色**: `#F8FAFC`（主文字）、`#94A3B8`（次要文字）、`#64748B`（辅助文字）
- **强调色**: `#22C55E`（成功）、`#F59E0B`（警告）、`#EF4444`（错误）

### 设计理念
- **视觉**: 沉浸式体验（深色主题）、渐变美学（深紫→青）、极简主义、层次分明
- **交互**: 直觉操作、流畅过渡、快捷操作、状态反馈

## 页面布局建议

### 全局布局（所有主页面共用）
- **布局模式**: 三栏式布局（左侧 Sidebar + 右侧主内容区 + 底部 Player Bar）
- **视觉重心**: 主内容区（随页面变化），底部播放栏常驻
- **侧边栏**: 固定宽度，深色背景，品牌Logo + 导航菜单 + 底部用户信息
- **底部播放栏**: 通栏固定，渐变毛玻璃效果，左侧歌曲信息、中间播放控制、右侧功能按钮

### 各页面布局要点

- **启动页**: 全屏居中布局，品牌Logo + 音频可视化动效 + 加载进度，背景紫青渐变流线装饰
- **登录注册页**: 左右双卡片布局（左侧品牌展示+耳机3D图，右侧登录表单），毛玻璃卡片效果
- **首页/发现页**: Banner轮播 + 推荐歌单横向滚动 + 热门歌手横向滚动
- **搜索结果页**: 顶部搜索框 + 分类Tab（全部/歌曲/歌手/专辑/歌单） + 卡片网格
- **我的音乐页**: 顶部Tab（本地音乐/我喜欢/下载歌曲/创建的歌单/收藏的歌单） + 歌曲列表
- **排行榜页**: 榜单分类Tab（领榜/热歌榜/新歌榜/原创榜） + 排行列表（带排名/头像/热度）
- **播放详情页**: 左右分栏（左侧黑胶唱片视觉，右侧歌词滚动列表）
- **歌单详情页**: 顶部歌单封面+信息+操作按钮 + 下方歌曲列表表格
- **歌手详情页**: 顶部歌手简介卡片（头像+介绍+关注按钮） + 下方热门歌曲列表
- **专辑详情页**: 顶部专辑封面+简介+播放全部按钮 + 下方曲目列表
- **歌词展示页**: 全屏毛玻璃窗口，居中歌词滚动，底部播放控制
- **用户个人中心页**: 多卡片网格布局（用户信息卡 + 数据统计卡 + 最近播放柱状图 + 我创建的歌单 + 收藏的歌单）
- **系统设置页**: 双层级侧边导航 + 右侧设置面板（开关/下拉/滑块等控件）
- **下载管理页**: 顶部进度条 + 下载列表表格（歌曲/歌手/速率/进度/操作）
- **播放队列页**: 播放队列列表 + 历史播放记录 + 清空操作

## 功能列表

### 全局功能
- **页面目标**: 提供统一的导航框架和播放控制，确保跨页面体验一致
- **功能点**:
  - **侧边栏导航切换**: 点击侧边栏菜单项切换主内容区页面，当前选中项高亮（左侧竖线 + 文字变色）
  - **底部播放栏常驻**: 所有页面底部显示播放控制栏，支持播放/暂停、上一曲/下一曲、进度拖拽、音量调节
  - **播放状态全局同步**: 当前播放歌曲、播放进度、音量等状态在所有页面间共享，切换页面不中断播放
  - **跳转播放详情**: 点击底部播放栏歌曲封面或展开按钮进入播放详情页

### 启动页
- **页面目标**: 品牌展示与加载动画，营造沉浸式第一印象
- **功能点**:
  - **品牌Logo展示**: 居中显示 Harmony Music Logo（紫青渐变 H 图标 + 品牌名）
  - **音频可视化动效**: Logo 下方展示声波频谱动画（青→蓝→紫渐变竖线跳动）
  - **加载进度指示**: 底部进度条展示加载进度，完成后自动跳转到登录页或首页
  - **背景流线装饰**: 左右两侧紫/青渐变流线装饰，营造音乐流动感

### 登录注册页
- **页面目标**: 用户账号入口，提供登录注册功能
- **功能点**:
  - **品牌展示卡片**: 左侧毛玻璃卡片展示品牌Logo和3D耳机渲染图，紫青渐变光效背景
  - **登录表单**: 邮箱/账号输入框 + 密码输入框 + 登录按钮（紫色渐变）
  - **注册入口**: 点击"注册"切换到注册表单模式
  - **登录跳转**: 点击登录按钮后跳转到首页/发现页

### 首页/发现页
- **页面目标**: 个性化推荐与内容发现
- **功能点**:
  - **Banner轮播**: 顶部运营Banner大图，带"去播放"按钮
  - **推荐歌单**: 横向滚动的歌单卡片列表，点击进入歌单详情页
  - **热门歌手**: 横向滚动的歌手头像卡片，点击进入歌手详情页
  - **顶部搜索框**: 输入关键词后跳转搜索结果页

### 搜索结果页
- **页面目标**: 搜索结果分类展示
- **功能点**:
  - **搜索输入**: 顶部搜索框，支持输入关键词实时搜索
  - **分类Tab切换**: 全部/歌曲/歌手/专辑/歌单 五个分类标签，点击切换结果类型
  - **结果卡片网格**: 以卡片网格形式展示搜索结果，点击进入对应详情页
  - **分页指示器**: 底部分页控件，支持翻页浏览

### 我的音乐页
- **页面目标**: 本地音乐与收藏管理
- **功能点**:
  - **分类Tab切换**: 本地音乐/我喜欢/下载歌曲/创建的歌单/收藏的歌单
  - **歌曲列表展示**: 表格形式展示歌曲（序号/封面/歌名/歌手/专辑/时长/操作）
  - **歌曲收藏/取消收藏**: 点击心形图标切换收藏状态，状态同步到 localStorage
  - **列表搜索过滤**: 列表顶部搜索框，支持在当前列表中过滤歌曲

### 排行榜页
- **页面目标**: 各类音乐榜单展示
- **功能点**:
  - **榜单分类Tab**: 领榜/热歌榜/新歌榜/原创榜 四个榜单分类切换
  - **排行列表展示**: 带排名数字、歌手头像、歌曲名、热度标签的排行列表
  - **前三名特殊样式**: 第1-3名使用奖牌/徽章装饰突出显示
  - **点击跳转**: 点击歌曲行播放歌曲，点击歌手名进入歌手详情页

### 音乐播放详情页
- **页面目标**: 核心播放控制界面，提供沉浸式播放体验
- **功能点**:
  - **黑胶唱片视觉**: 左侧拟物化黑胶唱片机，唱片随播放旋转
  - **歌词滚动显示**: 右侧歌词列表，当前播放行高亮（青蓝色），自动滚动跟随
  - **播放控制**: 播放/暂停、上一曲/下一曲、进度条拖拽、音量调节
  - **收藏/分享/下载**: 快捷操作按钮

### 下载管理页

## 数据共享配置

| 存储键名 | 数据说明 | 使用页面 |
|---------|---------|---------|
| `__global_harmony_currentSong` | 当前播放歌曲，类型 `ISong` | 所有页面（底部播放栏）、播放详情页、歌词页 |
| `__global_harmony_isPlaying` | 播放状态，类型 `boolean` | 所有页面（底部播放栏）、播放详情页、歌词页 |
| `__global_harmony_progress` | 当前播放进度（秒），类型 `number` | 所有页面（底部播放栏）、播放详情页、歌词页 |
| `__global_harmony_volume` | 音量大小（0-100），类型 `number` | 所有页面（底部播放栏）、设置页 |
| `__global_harmony_playMode` | 播放模式，类型 `'sequence' \| 'loop' \| 'random'` | 所有页面（底部播放栏）、播放队列页 |
| `__global_harmony_queue` | 播放队列，类型 `ISong[]` | 底部播放栏、播放队列页、播放详情页 |
| `__global_harmony_favorites` | 我喜欢的歌曲ID列表，类型 `string[]` | 我的音乐页、底部播放栏、各详情页 |
| `__global_harmony_user` | 当前用户信息，类型 `IUser` | 侧边栏、个人中心页、登录页 |
| `__global_harmony_recentPlay` | 最近播放记录，类型 `ISong[]` | 个人中心页、播放队列页 |

```ts
interface ISong {
  id: string;
  title: string;
  artist: string;
  artistId: string;
  album: string;
  albumId: string;
```

# UI 设计指南

## 1. 设计推导依据

- **参考意图**: Exact Reference —— 15 张 Harmony Music 高保真设计稿支配视觉风格、配色、布局结构与组件形态，文字内容按正常语义重建。
- **核心情绪 / 应用类型**: 沉浸式深色音乐播放器 PC 客户端，以紫青霓虹渐变传递流动的音乐韵律感。
- **独特记忆点**: 底部播放栏顶部贯穿一条紫→青渐变装饰线，进度条与播放按钮以青紫双色渐变呼应品牌，歌词页当前行带柔和紫光外发光。

## 2. Art Direction

- **方向名**: 紫青霓虹暗夜
- **Design Style**: Dark Frosted Glass + Gradient Neon —— 毛玻璃卡片、紫青渐变点缀、深色基底营造沉浸式音乐氛围，契合 PC 端音乐播放器的夜间使用场景。
- **DNA 参数**: 圆角 `rounded-xl`（卡片）/ `rounded-full`（按钮、进度滑块）；阴影 `shadow-lg` 柔光模糊；间距 `gap-4 / p-6`；字体方向 现代无衬线 + 几何感；装饰手法 渐变弥散光、音频波形线、霓虹描边。
- **应用类型**: Tool / Media Player —— 左侧固定导航 + 主内容区 + 底部常驻播放栏的三栏式桌面端布局。

## 3. Color System

**色彩关系**: 深靛蓝暗夜基底 + 青紫渐变主色（#6366F1 → #06B6D4）+ 亮青白文字，辅以毛玻璃半透卡片层。
**配色设计理由**: 深色背景降低夜间听歌视觉疲劳，紫青渐变作为品牌识别与主交互色贯穿按钮、进度条、激活态与装饰线；文字分三级确保长列表可读性；成功/警告/错误保持同饱和度冷调，不破坏暗夜氛围。
**主色推导**: 从设计文档指定的 #6366F1（深紫）与 #06B6D4（青色）提取渐变主轴，primary 取紫端作为品牌锚点，accent 取青端用于 hover 与次级高亮，进度条与装饰线使用双色渐变表达音乐流动感。
**使用比例**: 65% 深中性（bg/card/border）/ 25% 渐变辅助（accent/装饰光效）/ 10% primary（CTA、激活态、品牌）；主按钮、tab 激活、进度条、图标、边框不同时使用完整渐变。

| 角色 | CSS 变量 | Tailwind Class | HSL 值 | 设计说明 |
|---|---|---|---|---|
| bg | `--background` | `bg-background` | hsl(222 47% 11%) | 页面深靛蓝暗夜背景 |
| card | `--card` | `bg-card` | hsl(217 33% 17%) | 毛玻璃卡片、表单、弹层面板 |
| text | `--foreground` | `text-foreground` | hsl(210 40% 98%) | 标题与主文字 |
| textMuted | `--muted-foreground` | `text-muted-foreground` | hsl(215 20% 65%) | 次要说明、占位符、辅助元信息 |
| primary | `--primary` | `bg-primary` / `text-primary` | hsl(239 84% 67%) | 深紫主色，用于主按钮、激活导航、品牌标识 |
| primaryForeground | `--primary-foreground` | `text-primary-foreground` | hsl(0 0% 100%) | primary 上的白色文字图标 |
| accent | `--accent` | `bg-accent` | hsl(189 94% 43%) | 青色辅助色，用于 hover、进度点、播放按钮 |
| accentForeground | `--accent-foreground` | `text-accent-foreground` | hsl(0 0% 100%) | accent 上的白色文字图标 |
| border | `--border` | `border-border` | hsl(215 25% 27%) | 卡片、输入框、菜单边界，低对比 |

**语义色提示**: 成功 hsl(142 71% 45%) 三态 bg/border/text 均向冷绿偏移，用于下载完成、收藏成功；警告 hsl(38 92% 50%) 暖橙金，饱和度与 primary 对齐 ±10%，用于下载暂停、榜单热度；错误 hsl(0 84% 60%) 冷调红，用于下载失败、通知红点。三者饱和度控制在 60-75%，避免霓虹刺眼。

## 4. 字体与节奏

- **font-display**: Space Grotesk + Noto Sans SC —— 几何感无衬线强化科技音乐气质，中文保持清晰可读。
- **font-body**: Inter + Noto Sans SC —— 长列表、歌词、设置页保证长时间阅读舒适度。
- **字号**: H1 text-4xl ~ text-5xl；H2 text-xl ~ text-2xl；body text-sm ~ text-base；muted text-xs。
- **圆角**: 大圆角 —— 卡片 `rounded-xl`、按钮 `rounded-lg`、播放按钮 `rounded-full`，呼应音乐的柔和流动感。

## 5. 全局布局契约

- **Reference Layout Use**: 左侧固定侧边导航（宽 240px）+ 顶部搜索/用户栏 + 主内容滚动区 + 底部常驻播放栏（高 72px）的三栏一底结构完全来自参考图；各页面内部模块顺序与信息密度对齐对应设计稿。
- **Page / Section Order**: 启动页 → 登录注册 → 首页/发现 → 播放详情 → 歌单详情 → 搜索结果 → 我的音乐 → 歌手详情 → 专辑详情 → 排行榜 → 全屏歌词 → 个人中心 → 系统设置 → 下载管理 → 播放队列，共 15 页。
- **Standard Content Zone**: 主内容区 `max-w-[1400px] mx-auto px-6 py-8`，PC 端宽屏适配。
- **Shell / Frame Alignment**: 侧边栏与底部播放栏为固定 chrome，内容区独立滚动，与框架同宽对齐。
- **Padding & Rhythm**: `px-6 py-8`，区块间距 `gap-8`，卡片内距 `p-6`，保持 8px 倍数。
- **Full-bleed Zones**: 启动页、登录页背景渐变全宽；首页 Banner、歌词页背景可全宽，内部操作按钮受内容区约束。
- **Local Narrowing**: 登录表单、设置页正文可收窄至 `max-w-2xl` 居中。
- **Overflow Strategy**: 歌曲列表、排行榜、下载管理表格使用 `overflow-x-auto`；横向歌单/歌手滚动使用 `overflow-x-auto` + 隐藏滚动条。
- **Flexibility Boundary**: 允许移动端收窄侧边栏为图标模式、调整卡片列数；不允许改变主色、渐变方向、圆角系统和底部播放栏结构。

## 6. 视觉与动效

- **装饰**: 紫青弥散光晕、音频波形线、渐变装饰线
- **阴影/边界**: 中 —— 卡片柔光阴影 + 1px 半透明白边模拟毛玻璃边缘
- **动效**: 精致 —— 播放按钮缩放、进度条平滑过渡、导航激活渐变滑入、歌词行淡入高亮、黑胶唱片旋转

## 7. 组件原则

- 按钮、输入框、菜单项、卡片必须有 Default / Hover / Active / Focus-visible / Disabled 五态。
- Primary 按钮用紫色实底，播放主操作用青色圆形按钮，二者权重分离。
- 进度条、滑块、开关使用紫→青渐变填充，thumb 为青色圆形带微光。
- 导航激活态使用左侧青色竖线 + 浅紫背景，不使用整行渐变。
- 空状态、加载态延续深色 + 渐变骨架屏，不回退默认 shadcn 浅色样式。

## 8. Image Direction

- **Image Role**: 歌单封面 / 专辑封面 / 歌手头像 / Banner 主视觉 / 登录页品牌插画
- **Image Art Direction**: 霓虹暗夜风格，紫青双色打光，人物侧光或逆光轮廓，背景虚化带光斑；封面构图居中主体，底部留 20% 暗区便于叠加文字；整体色调偏冷紫青，避免暖黄橙主导；质感偏胶片颗粒 + 轻微柔焦。
- **Image Prompt Keywords**: neon portrait, purple and cyan lighting, dark background, cinematic rim light, music artist, vinyl record aesthetic, grain texture, moody atmosphere, synthwave color grade, soft bokeh
- **Image Avoidance**: 电商素材感歌手图、明亮白日外景、过度磨皮网红风、无主题抽象渐变图、低分辨率模糊封面、暖黄色调主导图

## 9. Anti-patterns

- **Split personality**: 页面之间切换主色、渐变方向或圆角大小；全站共享同一套紫青渐变系统与圆角语言。
- **Gradient everywhere**: 渐变铺满按钮、卡片、边框、文字、图标；按 65-25-10 比例把渐变收回到装饰线、进度条、主 CTA。
- **Default dark drift**: 回到纯黑 + 灰的通用深色后台；用深靛蓝基底 + 青紫点缀建立音乐产品辨识度。
- **Invisible interaction**: hover 做了渐变，focus-visible 丢了；每个可交互元素都要有键盘可见的青色描边状态。
- **Low contrast lyrics**: 歌词页非当前行透明度过低无法阅读；未播放行保持 ≥ 3:1 对比度，当前行用发光强化。
- **Status color drift**: 成功/警告/错误饱和度过高刺眼；语义色饱和度与 primary 对齐 ±15%，保持冷调暗夜统一感。
